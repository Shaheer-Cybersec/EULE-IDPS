# Python Automation Scripts

This directory contains Python scripts for automated threat response and alert processing.

## Planned Scripts

- `alert_parser.py` - Parse Wazuh JSON alerts
- `auto_response.py` - Orchestrate automated responses
- `ip_blocker.py` - Automated firewall IP blocking
- `process_killer.py` - Terminate malicious processes
- `notify.py` - Send incident notifications

## Requirements
Python 3.10+ required. Dependencies listed in `requirements.txt`

*Scripts will be developed during Phase 3*
